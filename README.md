\# Pizza Sales Analysis Project



\## Overview

This project analyzes pizza sales data to identify trends in revenue, popular pizza types, and order timing.



\## Database Structure

The project uses four tables:

\- Orders: Tracking the date and time of each purchase.

\- Order Details: Linking orders to specific pizzas and quantities.

\- Pizzas: Storing sizes and prices.

\- Pizza Types: Categorizing pizzas (e.g., Veggie, Classic).



\## Key Queries Included

\- Total Revenue Calculation

\- Most common pizza size ordered

\- Top 5 most ordered pizza types

\- Revenue contribution by category



\## Dataset Information

The datasets used in this project are located in the `/data` folder. 

\- Orders.csv: Contains order ID, date, and time.

\- Order\_Details.csv: Contains the specific pizzas and quantities for each order.

\- Pizzas.csv: Contains price and size information.

\- Pizza\_Types.csv: Contains names and categories.
